# CNTV+ AX JOURNEY — D1 진단 리포트 HTML 구현 플랜

**Goal:** D1 인터뷰 6명 진단 결과를, 06 질문지와 동일한 디자인 시스템의 standalone HTML 리포트 1장으로 만든다. 참가자+팀장이 읽는 성장 피드백 톤, AX 단계 실명 공개.

**Architecture:** SSOT(`curriculum/cntv-d1-인터뷰-진단.md`)의 내용을 누락 없이 lift → `materials/07-cntv-ax-d1-진단리포트.html`. 06 질문지(`materials/06-cntv-ax-인터뷰-질문지.html`)의 CSS 토큰·컴포넌트를 베이스로 재사용하고, 리포트 전용 컴포넌트(격차 바 차트·개인 진단 카드·로드맵)를 추가한다. 외부 의존성은 06과 동일(Pretendard CDN, JetBrains Mono).

**Tech Stack:** 순수 HTML/CSS + 최소 vanilla JS(scroll-reveal, nav progress). 빌드 없음, 브라우저에서 직접 open.

## Global Constraints (전부 확정된 결정 · 변경 시 사용자 승인)

- 파일: `materials/07-cntv-ax-d1-진단리포트.html` (06과 짝 → 07번)
- 디자인 시스템: `.claude/rules/design-system.md` + 06 질문지 준수. 공통=앰버·오렌지(`--grad-common`), 심화=블루·바이올렛(`--grad-adv`), 기초=에메랄드(`--grad-basic`)
- 폰트: Pretendard(본문) / JetBrains Mono(라벨·수치). 픽셀·레트로 폰트 금지
- 톤: **성장 피드백형** — "이미 여기까지 왔다 → 남은 건 여정" 격려 프레이밍. 냉정한 gap-shaming 금지
- 공개: **AX 단계 실명 전체 공개**. 6명 실명 카드 + 단계
- 박성희 실장: **녹취 누락** → 최소 정보 + "인터뷰 보강 예정" 뱃지. 추정치는 회색·점선 처리
- 언어: 한국어. 영어 용어는 `<span class="en-note">` 관용
- 반응형(`max-width:768px`) + 인쇄(`@media print`, A4) 필수
- 사실 정확성: 수치·고유명사는 SSOT와 1:1 일치(예: 447건, 세일코리아넷 10일치, 프로플랜 3만원, 3사 법인 신지·2C지·심플레이)
- 커밋: 완료·검수 후 사용자 확인받고 (개인 동기화 레포라 main 직푸시 OK)
