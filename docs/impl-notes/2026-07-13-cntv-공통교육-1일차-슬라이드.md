# CNTV+ 공통교육 1일차 — HTML 슬라이드 덱

- 산출물: `materials/10-cntv-ax-공통교육-1일차.html` (47장, standalone, 1280×720 scale-to-fit)
- index.html 카드 추가 완료. Playwright 스크린샷으로 6개 대표 슬라이드 렌더링 검증 완료.

## 확정 사항 (사용자 취조)

- 형식 = **HTML 슬라이드 덱** (pptx 아님). 키보드 ←→/스페이스, 클릭 버튼, 스와이프, `#n` 해시 내비게이션, 인쇄 대응.
- 1일차 범위 = **조직방향 → LLM 기본 개념 → 세팅**. 2일차 = 스킬·자동화 → 수미상관("조직 차원의 AX는 차원이 다르다") 마무리.
- 톤 = 현행 유지 (기회·순서의 언어, 감성 레벨 3).
- 수미상관 씨앗: 1일차 마지막 슬라이드(47)가 "개인의 AX를 넘어 조직의 AX로 — 결말은 2일차 끝에서"로 2일차 피날레를 예고.

## 구성 (47장)

오프닝 4 (병목은 "나" 훅 + 07 분포 밴드) → 1부 조직방향 30 (비즈니스 이해 4 · 변화적응력 4 · 조직문화: 자가진단 8문 각 1장 + 늙크크 7문 각 1장 + 영크크 1장 + 5원칙 + 종착지) → 2부 원리 6 (예측기계·프롬프트·컨텍스트·환각검증·보안습관) → 3부 세팅 4 → 클로징 3.

## ⚠️ DEVIATION / 판단 (보수적 선택, 검토 요망)

1. **늙크크 = 7문 채택** — 대화에서 받은 최신 원문(7개)이 SSOT `cntv-ax-journey.md` 부록 A(5개 구버전)와 다름. **최신 대화 원문을 채택**했고 SSOT 부록 A는 아직 v3로 갱신하지 않음 → 사용자 확인 후 SSOT 동기화 필요.
2. **보안 습관 슬라이드(40)를 1일차 2부 끝에 배치** — 07 매핑 테이블 순서(원리·세팅·스킬화·봇·보안)상 보안이 마지막이지만, 개념적으로 환각·검증과 붙는 게 자연스러워 1일차에 배치. 2일차로 이동 가능.
3. **영크크는 4문 한 장** — 원문 목차에 자가진단·늙크크만 "한 질문당 한 페이지" 마킹이 있어 문자 그대로 따름.
4. 늙크크 5번은 원문상 역방향 자기점검("찔리는 느낌이 먼저 든다")이라 질문형으로 바꾸고 "이번엔 반대로" 태그 부착.
5. 슬라이드 3(분포 밴드)은 07 리포트의 무기명 조직 단위 표현·톤 기준을 그대로 따름.

## 기술 메모

- Pretendard variable CDN + repo 디자인 시스템 팔레트(하우스 컨벤션 준수). 문서형 규칙(TOC·Chapter)은 슬라이드 포맷 특성상 미적용 — 콘텐츠 규칙(언어·톤)만 적용.
- 스크린샷 검증 시 등장 애니메이션(.rv, ~0.8s) 때문에 `--wait-for-timeout=1600` 필요.
