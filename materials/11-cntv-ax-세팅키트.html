AI-작업폴더/ ← 루트 (모든 업무 공통) ├── CLAUDE.md ← 공통 규칙: 나 · 회사 · 말투 └── 정산-업무/ ← 업무 하나당 폴더 하나 ├── CLAUDE.md ← 이 업무만의 규칙 ├── rules/ ← 규칙이 길어지면 주제별로 │ ├── 양식.md │ ├── 용어.md │ └── 금지사항.md ├── 재료/ ← 원본 엑셀 · 양식 · 예시 │ └── 일계표_샘플.xlsx ← 실습용 (가상 데이터) └── 진행노트.md ← 상태(지도)

어디에 두나 — Windows / Mac

찾기 쉬운 문서 폴더를 권합니다. [내이름]은 실제 사용자명으로.

🪟 Windows
C:\Users\[내이름]\Documents\AI-작업폴더

탐색기 주소창에 %USERPROFILE%\Documents

Mac
/Users/[내이름]/Documents/AI-작업폴더

Finder → 이동 → ~/Documents

⚠️ Google Drive · iCloud · Dropbox 같은 자동 동기화 폴더 안에는 두지 마세요. 동기화가 파일을 건드리는 순간 폴더가 꼬일 수 있습니다. (원본 데이터만 클라우드에, 작업 폴더는 별도로.)

어떻게 구성했나

받으면 바로 돌아가게 CNTV+ 예시로 채워뒀습니다. [ ] 표시된 개인정보만 본인 걸로 바꾸면 끝.

  1. 루트 CLAUDE.md — 코워크가 일하는 방식(인터뷰 → 레퍼런스 → 노트 → 확인 → 암묵지 수집, 이탈 시 재확인) + 회사·말투
  2. 정산-업무/CLAUDE.md — 일계표 데이터 구조와 결과물 규칙(실매출·마진 기준, 채널별·카테고리별)
  3. rules/ — 양식(날짜·금액·채널명 통일) · 용어(실매출·FM4·세일코리아넷 등) · 금지사항(가드레일)
  4. 재료/ — 실습용 일계표_샘플.xlsx(가상 데이터). 여기에 실제 원본을 넣으면 된다
  5. 진행노트.md — 결정·진행·남은 일을 적는 "지도". 코워크가 근황을 이어받는다
💡 일하다 "이건 매번 말하네" 싶은 규칙이 나오면 — 코워크가 먼저 "CLAUDE.md에 저장할까요?"라고 물어봅니다. 승인하면 스스로 적습니다.
