# CNTV+ AX 세팅 키트 — 시작 안내

> 클로드 코워크에게 "우리 회사"를 알려주는 기본 폴더입니다.
> 여러분은 **대괄호 `[ ]` 부분만 여러분 정보로 채우면** 됩니다. 나머지는 그대로 두세요.

---

## 1. 이 키트가 하는 일

교육에서 배운 **"AI에게 심는다"의 세 층**을 폴더로 만들어 둔 것입니다.

| 층 | 이 키트의 파일 | 무엇 |
|---|---|---|
| **규칙** (AI의 성격) | `CLAUDE.md`, `rules/` | 내가 누구고, 우리 회사 규칙은 뭔지 |
| **지식** (AI의 기억) | `재료/` 폴더 | 원본 엑셀·양식·잘 된 예시를 넣는 곳 |
| **상태** (AI의 근황) | `진행노트.md` | 뭐가 결정됐고, 어디까지 왔나 |

코워크는 이 폴더를 여는 순간 위 파일들을 **자동으로 읽고** 일을 시작합니다.

---

## 2. 폴더 구조

```
AI-작업폴더/                     ← 루트 (모든 업무 공통)
├── CLAUDE.md                   ← 공통 규칙: 나 · 회사 · 말투
└── 정산-업무/                   ← 업무 하나당 폴더 하나
    ├── CLAUDE.md               ← 이 업무만의 규칙
    ├── rules/                  ← 규칙이 길어지면 주제별로 쪼갠다
    │   ├── 양식.md
    │   ├── 용어.md
    │   └── 금지사항.md
    ├── 재료/                    ← 원본 엑셀 · 양식 · 예시를 넣는 곳
    └── 진행노트.md              ← 상태(지도): 결정 · 진행 · 남은 일
```

- 업무가 여러 개면 `정산-업무/`를 복사해서 `보고서-업무/`, `CS-업무/`처럼 늘리면 됩니다.
- 규칙은 **가까운 것이 우선**입니다. 루트 `CLAUDE.md` 위에 업무 폴더 `CLAUDE.md`가 우선.

---

## 3. 어디에 두나 — 경로 예시 (Windows / Mac)

폴더는 아무 데나 둬도 되지만, 찾기 쉬운 곳을 권합니다.

**Windows**
```
C:\Users\[내이름]\Documents\AI-작업폴더
```
(문서 폴더 안. 탐색기 주소창에 `%USERPROFILE%\Documents` 치면 바로 열립니다.)

**Mac**
```
/Users/[내이름]/Documents/AI-작업폴더
```
(Finder → 이동 → `~/Documents`)

> 팁: 한글 폴더명도 되지만, 나중에 배포·공유까지 갈 거면 영문(`ai-work`)이 안전합니다.

> ⚠️ **주의:** 이 폴더를 Google Drive · iCloud · Dropbox 같은 **자동 동기화 폴더 안에는 두지 마세요.** 동기화가 파일을 건드리는 순간 폴더가 꼬일 수 있습니다. (원본 데이터만 클라우드에, 작업 폴더는 별도로.)

---

## 4. 채우는 순서 (5분)

1. `CLAUDE.md` 열어서 `[ ]` 부분을 내 정보로 채운다 (이름 · 부서 · 회사 · 말투).
2. `정산-업무/`를 내 업무 이름으로 바꾸고, 그 안 `CLAUDE.md`도 채운다.
3. `재료/` 폴더에 실제 엑셀·양식·잘 된 예시 결과물을 넣는다.
4. 코워크로 이 폴더를 열고 — 일을 시켜본다.
5. 일하다 "이건 매번 말하네" 싶으면 → **"방금 그 규칙 CLAUDE.md에 추가해줘"** 한마디. (AI가 스스로 적습니다.)

준비 끝. 이제 코워크가 여러분의 일을 아는 동료가 됩니다.
