# 재료 폴더 (AI의 지식)

이 폴더에 코워크가 참고할 **원본 재료**를 넣으세요.

## 넣으면 좋은 것

- 📊 **원본 엑셀** — 채널별 주문·매출 파일 (예: `일계표_샘플.xlsx`)
- 📄 **양식 파일** — "이런 모양으로 만들어줘" 하는 빈 템플릿
- ⭐ **잘 된 예시 결과물** — 지난달에 잘 만든 현황판. **예시 하나가 설명 열 줄보다 강합니다.**

## 넣지 말 것

- ❌ 계좌·개인정보 등 민감한 원본 (금지사항.md 참고)
