# 용어 사전

> 우리 팀·홈쇼핑에서 쓰는 말을 코워크에게 알려주는 곳. AI가 헷갈리지 않게 됩니다.

| 우리 말 | 뜻 |
|---|---|
| 실매출 | 매출액 − 반품액 (반품 뺀 진짜 매출) |
| 마진 / 마진총액 | 실매출 − 매입총액 |
| 마진율 | 마진 ÷ 실매출 × 100 (%) |
| 채널 | 상품을 판 곳 (네이버·쿠팡·세일코리아 등) |
| 자사 상품 / 타사 상품 | 우리가 소싱한 상품 / 외부 업체 상품 |
| 자사몰 | 우리 회사 직영 온라인몰 (FM4·플라이마트로 운영) |
| 일계표 | 하루치 채널·상품별 매출을 정리한 표 |
| FM4 (플라이마트) | 자사몰 주문취합·발주·정산 시스템 |
| 세일코리아넷 | 외부 판매 채널 (주문·매출 데이터 제공, 10일치씩 다운) |
| 편성매출 | 방송 편성 기준으로 잡은 자사 매출 (세일코리아 주문매출과 매핑 필요) |
| 매핑 테이블 | 세일코리아 상품명 ↔ 우리 편성명을 이어주는 대조표 |
