# 양식 규칙

> 결과물을 만들 때 지킬 형식입니다.

## 날짜 · 숫자

- 날짜: `YYYY-MM-DD` (예: 2026-07-14)
- 금액: 천 단위 콤마 + "원" (예: 1,250,000원)
- 비율(마진율 등): 소수 첫째 자리 + "%" (예: 34.2%)
- 수량: 콤마 + "개" (예: 1,200개)

## 채널 이름 표기 (통일)

들쭉날쭉한 채널명은 아래로 통일한다.

| 이렇게 들어와도 | 이렇게 표기 |
|---|---|
| 십일번가 / 11st | **11번가** |
| 세일코리아넷(채널) / (CN) / (씨네) | **세일코리아** |
| 자사몰 / FM4 / 플라이마트 | **자사몰** |
| G마켓 / 지마켓 | **지마켓** |

## 결과물 구성 순서

1. **총매출 · 총마진 요약** (맨 위, 큰 숫자)
2. **채널별 매출 표** (실매출 기준, 매출 높은 순)
3. **카테고리별 비중** (식품/주방/생활/뷰티/패션잡화)
4. **상품 매출 TOP 10**
5. **채널별 마진율** 비교

## 파일 이름

- 결과물: `결과-YYYYMMDD_채널별매출현황.xlsx`
- 중간 작업본: `작업-YYYYMMDD.xlsx`
